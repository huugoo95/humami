# HUMAMI.md — Operating System del proyecto

> Fuente de verdad operativa para trabajar con tenacitas sin repetir contexto en cada mensaje.

## 1. Objetivo actual
Convertir Humami en un producto monetizable centrado en **resolución semanal de cenas** (rápidas y anti-desperdicio), manteniendo la identidad gastronómica premium de marca.

## 2. Prioridades (Now / Next / Later)

### NOW
- Definir propuesta comercial concreta (qué se vende, a quién y a qué precio).
- Mejorar producto base (plan semanal + lista de compra + reutilización de ingredientes).
- Preparar identidad visual inicial (logo + sistema mínimo de marca).
- Crear pipeline de contenido (artículos y piezas largas cortas, incluyendo mini-libros).

### NEXT
- Automatizar creación/publicación de recetas desde API.
- Preparar landing y flujo de validación de pago.
- Estandarizar assets de marca y plantillas editoriales.

### LATER
- Escalar distribución y partnerships.
- Explorar líneas B2B y producto físico.

# Task Closure Checklist

Before marking any task as done:

- [ ] Task one-liner exists (`[TRACK] objective | done-criteria | deadline`)
- [ ] Acceptance criteria explicitly checked
- [ ] Validation evidence available (tests/logs/smoke/screens)
- [ ] PR/workflow conventions respected
- [ ] Operational docs updated if needed:
  - [ ] `ops/decisions.md`
  - [ ] `ops/incidents.md`
  - [ ] relevant runbook (`ops/runbook-*.md`)
- [ ] Next step identified (or explicitly `none`)
